2. FAQ Accordion Does Not Expand
Issue:
Your code is almost correct, but you have custom jQuery code that interferes with Bootstrap’s default accordion behavior:

javascript
Copy
Edit
$('[data-toggle="collapse"]').on('click', function(e) {
    e.preventDefault();
    const target = $(this).attr('data-target') || $(this).attr('href');
    $(target).collapse('toggle');
    // Update aria-expanded attribute
    const isExpanded = $(target).hasClass('show');
    $(this).attr('aria-expanded', !isExpanded);
});
Problem:

This is unnecessary—Bootstrap's collapse already works by default!

Your code prevents the default behavior, so accordion never expands/collapses properly.

Solution:
Remove or comment out this block:
