Missing Initial CSS: FAQ Answers Should Be Hidden by Default
If you don’t hide .faq-answer in CSS, they’re always visible, so toggling won’t work.
If they’re set to display:none !important, JS might not be able to show them.

Add this to your CSS (if it’s not there):

css
Copy
Edit
.anaheim-faq-list .faq-answer {
  display: none;
}
(If you want the first FAQ open, make just that one display:block with a class.)

2. FAQ jQuery: Your code is fine!
You have:

js
Copy
Edit
$('.faq-question').on('click', function() {
    const $item = $(this).closest('.anaheim-faq-item');
    const $answer = $item.find('.faq-answer');
    const $toggle = $(this).find('.faq-toggle i');
    
    // Close others (accordion)
    $('.anaheim-faq-item').not($item).each(function() {
        $(this).find('.faq-answer').slideUp(300);
        $(this).find('.faq-toggle i').removeClass('fa-minus').addClass('fa-plus');
    });
    
    // Toggle current
    $answer.slideToggle(300);
    $toggle.toggleClass('fa-plus fa-minus');
});
This is 100% correct.
