How to Show the Full Image
1. Remove or Increase max-height on the image
In your CSS:

css
Copy
.why-choose-image img {
  width: 100%;
  height: auto;               /* This will maintain aspect ratio */
  object-fit: contain;        /* Shows the whole image inside the box */
  max-width: 600px;           /* You can keep or increase this */
  /* Remove or increase this: */
  /* max-height: 400px; */
  border-radius: 8px;
}
Try removing max-height: 400px; or set it to a higher value, like max-height: 500px; or 600px;
You can also try just removing it altogether if the parentâ€™s size is fine.

2. Allow the Container to Grow with the Image
You might also want the image container to adjust:

css
Copy
.why-choose-image {
  flex: 0 0 50%;
  display: flex;
  align-items: center;
  justify-content: center;
  overflow: visible;    /* Allow overflow to see the full image */
  max-width: 50%;
  padding: 20px;
  min-height: 0;        /* Allow it to shrink if needed */
}
3. For Best Fit, Try This Combo:
css
Copy
.why-choose-image img {
  width: 100%;
  height: auto;            /* Maintain natural aspect ratio */
  object-fit: contain;     /* Show entire image */
  max-width: 100%;         /* Make it scale with container */
  border-radius: 8px;
}
Remove any hard max-height unless you want a strict limit.

If you want to make sure the whole image is always visible, use only width: 100%; height: auto; object-fit: contain;.

Quick Solution (Replace Your CSS With This):
css
Copy
.why-choose-image img {
  width: 100%;
  height: auto;
  object-fit: contain;
  max-width: 100%;
  border-radius: 8px;
}
If You Want a Maximum Height But No Cropping:
css
Copy
.why-choose-image img {
  width: auto;
  max-width: 100%;
  max-height: 500px;   /* or remove this line for no height limit */
  object-fit: contain;
  border-radius: 8px;
}
Summary Table
Problem	Solution
Image cut off/cropped	Use height: auto; object-fit: contain; and remove hard max-height
Image too tall	Add max-height: 600px; (optional)
Image too small	Increase container width, or increase max-width on image
