2. FAQ Not Expanding
Common Issues:
The FAQ section uses Bootstrap collapse, but the proper JS/CSS isn't loaded.
IDs and data-bs-target/href attributes don't match.
FAQ JS is missing or conflicts with other code.
Solution Example:
